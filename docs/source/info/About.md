# About

PyMDL is a webscra-ping API made for the website [MyDramaList](https://mydramalist.com/) (MDL).

## Libraries Used
- [beautifulsoup4](http://www.crummy.com/software/BeautifulSoup/bs4/)
- [requests](https://requests.readthedocs.io/)
- [lxml](https://lxml.de/)
